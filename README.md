# Coffee Shop Sales Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on coffee shop sales data to uncover business insights, customer purchasing patterns, product performance, and sales trends.

The objective is to analyze sales transactions and identify key factors that influence revenue and customer behavior using Python and data visualization techniques.

---

## Dataset

The dataset contains coffee shop transaction and sales records.

### Features Included

* Transaction ID
* Product Category
* Product Type
* Product Name
* Quantity Sold
* Unit Price
* Sales Amount
* Transaction Date
* Transaction Time
* Store Location

---

* Jupyter Notebook

* Seaborn

## Code Implementation

### Data Loading

The coffee shop sales dataset was loaded using Pandas for analysis.

### Data Exploration

Initial inspection was performed using:

* head()
* info()
* describe()
* value_counts()

to understand dataset structure and statistics.

### Data Cleaning

* Checked for missing values
* Removed inconsistencies
* Formatted date and time fields
* Prepared data for analysis

### Exploratory Data Analysis (EDA)

Analyzed sales transactions and product performance using statistical methods and visualizations.

### Sales Performance Analysis
